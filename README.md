# Comprehensive Portal for Analyzing ADX Market Bulletin

## Overview
This project is designed to create a user-friendly web portal for analyzing the Abu Dhabi Exchange (ADX) Market Bulletin for 30th April 2026. The portal provides dynamic data visualization, trend analysis tools, and downloadable datasets in user-friendly formats (XLSX and CSV).

## Features
1. **Dynamic Data Visualization:** Pre-built graphs for analyzing stock trends, sector performance, and market capitalization distributions.
2. **Trend Analysis Tools:** Identify patterns in trading volumes, price movements, and sector performance.
3. **Search and Filter Options:** Locate specific data based on company symbols, sectors, or market capitalization thresholds.
4. **Interactive Dashboards:** Display key metrics for immediate insights.
5. **Download Options:** Download datasets in XLSX and CSV formats for offline analysis.
6. **API Integration:** Allows developers to integrate the data into their custom analytical tools.

## Installation
1. Clone the repository from GitHub:
   bash
   git clone https://github.com/username/adx-market-bulletin-analyzer.git
   cd adx-market-bulletin-analyzer
   
2. Install the required Python packages:
   bash
   pip install -r requirements.txt
   
3. Place the `ADX_Market_Bulletin_30th_April_2026.xlsx` dataset in the project directory.

## Running the Application
1. Start the Flask server:
   bash
   python app.py
   
2. Open your web browser and navigate to `http://127.0.0.1:5000/`.

## Usage
- Select a sector and/or company from the dropdown menus.
- Click the 'Filter' button to generate a visualization.
- View the generated chart on the results page.
- Download the dataset for offline use (XLSX/CSV formats).
